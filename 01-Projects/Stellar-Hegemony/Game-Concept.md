# Game Concept — Stellar-Hegemony

## High Concept
A nation-building / management sim inspired by *Rumble Nation*, focused on political intrigue, resource management, and emergent storytelling.

## Core Pillars
- **Political Power** — Manage factions, alliances, and rivalries
- **Resource Economy** — Deep but readable production chains
- **Character-Driven** — Leaders and advisors with personalities and agendas

## Initial Scope
- 2D top-down / isometric view
- Single-player campaign + sandbox mode
- Mod-friendly from the start

*More details to be expanded as the design solidifies.*
