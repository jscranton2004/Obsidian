# Stellar Hegemony - Phase 5: Content & Balance

**Date:** 2026-05-30  
**Status:** Planning

## Goal
Finalize content, improve balance, and prepare the game for broader testing and eventual release.

## Current State
- Core gameplay loop is complete and polished
- All major systems (Leader Selection, Deployment, Movement, War Phase) are functional
- The game is playable but has not yet been deeply balanced or tested at scale

## Phase 5 Objectives

1. **Faction & Leader Balance**
   - Review and balance the 6 factions and their leaders
   - Ensure no faction is significantly stronger or weaker

2. **Deployment & Zone Value Tuning**
   - Review the 2–12 zone value system
   - Adjust values or deployment rules if needed for better flow

3. **Playtesting & Edge Cases**
   - Test with multiple players and game lengths
   - Identify and fix edge cases and rule ambiguities

4. **Documentation & Onboarding**
   - Improve in-game clarity and external documentation
   - Prepare basic rules summary or tutorial elements

## Recommended Task Order

| Order | Task | Scope | Priority |
|-------|------|-------|----------|
| 1     | Initial faction and leader balance review | Medium | High |
| 2     | Deployment and zone value tuning | Medium | High |
| 3     | Playtesting pass and edge case fixes | Medium | High |
| 4     | Documentation and onboarding improvements | Medium | Medium |

## Out of Scope (for this phase)
- New gameplay mechanics
- Advanced AI
- Full Steam release polish (art, sound, marketing)

## Success Criteria
- The game feels balanced across multiple playthroughs
- Common edge cases are handled cleanly
- New players can understand the core loop without major confusion
